
出现提示时，输入您的 [Z.ai](http://Z.ai) API 密钥。请向黑客马拉松组织者 **Jesse** 索取您的密钥。

### 响应缓慢

如果 Claude Code 响应缓慢，可能是因为正在处理较大的代码库。请尝试：

- 在较小的子目录中启动 Claude Code
- 使用更具体的提示词，减少对整个项目的搜索范围
- 运行 `/clear` 重置过长的对话上下文
